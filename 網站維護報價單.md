## 📋 系統概述

本系統為專業的德州撲克協會專用賽事管理平台，採用現代化技術架構開發，提供完整的賽事管理、獎金計算、數據統計等功能。

### 技術架構
- **前端框架：** React 18 + TypeScript
- **樣式系統：** Tailwind CSS
- **雲端服務：** Firebase (數據存儲、認證)
- **部署平台：** Vercel
- **構建工具：** Vite
