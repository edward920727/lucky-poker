# 快速启动指南

## 第一步：安装依赖

在项目根目录运行：

```bash
npm install
```

## 第二步：启动开发服务器

```bash
npm run dev
```

启动后，浏览器会自动打开（通常是 http://localhost:5173）

## 第三步：使用系统

1. **选择赛事类型**
   - 在首页选择要举办的赛事（600/1200/3400/6600/11000）

2. **新增玩家**
   - 点击「會編」输入框，会弹出虚拟数字键盘
   - 输入会编（例如：A105）
   - 如果该会编有历史纪录，会自动弹出历史信息
   - 输入玩家名称
   - 点击「新增玩家」

3. **管理玩家**
   - 在玩家列表中可以看到所有已登记的玩家
   - 可以增加买入次数（点击 +1 按钮）
   - 可以修改当前码量
   - 可以移除玩家

4. **查看统计**
   - 顶部统计面板实时显示：
     - 參賽人數
     - 理論總碼量
     - 實際登記總碼量
   - 如果码量不符，会显示红色警告

5. **导出结算表**
   - 点击右上角「📥 導出結算結存表」按钮
   - 系统会生成 PNG 图片并自动下载

## 测试数据

系统预设了以下测试会编的历史数据：
- **A105**: 有 6600 和 3400 赛事的历史纪录
- **B203**: 有 1200 赛事的历史纪录

输入这些会编时，会自动显示历史信息。

## 常见问题

**Q: 如何修改赛事类型和起始码量？**
A: 编辑 `constants/pokerConfig.ts` 文件中的 `TOURNAMENT_TYPES` 对象。

**Q: 如何添加更多玩家历史数据？**
A: 在 `constants/pokerConfig.ts` 的 `PLAYER_HISTORY_DB` 对象中添加数据。

**Q: 数据会保存吗？**
A: 当前版本使用内存存储，刷新页面会丢失数据。如需持久化，可以：
- 使用 localStorage
- 连接后端 API
- 使用数据库

**Q: 支持哪些会编格式？**
A: 支持字母和数字组合，例如：A105, B203, C001 等。
